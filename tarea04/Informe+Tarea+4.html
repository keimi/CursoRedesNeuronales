Informe Tarea¶

Introducción¶

El objetivo de esta tarea es programar el cdigo necesario para realizar experimentos de programacin genetica y probar el cdigo en ejemplos simples implementados para este propsito.

Los experimentos desarrollados para esta tarea fueron:

  • Algoritmo de programacin genetica para obtener el mximo entre dos ecuaciones
  • Algoritmo de programacin genetica que simule el programa Des Chiffres et des Lettres

Experimentos¶

Máximo entre dos ecuaciones¶

Para realizar este experimento se progamaron dos clases en el archivo abstract_tree.py, la clase Node y AbstractTree. Ambas clases se utilizaran luego como padres para implementar las clases para el segundo experimento.

Tambien se utiliz el mismo algoritmo gentico implementado para la tarea 3, con algunos ajustes para que aceptara arboles en vez de vectores. Se tuvo que implementar una clase Fit para cada experimento.

import numpy as np
import matplotlib.pyplot as plt
from genetic import Genetic
from genetic import FitAbstractTree

max_depth = 3
pop_size = 100
max_generation = 300

# clase de Fit especializada para el problema espcfico.

asbtract_tree = Genetic(fitness_class=fit_class,
                        gene_size=max_depth, 
                        population_size=pop_size, 
                        mut_rate=0.01,
                        max_fitness=1E100,
                        n_pool=None)

fitest, best_vector, mean_vector = asbtract_tree.evolve(max_generation)

“Des Chiffres et des Lettres”¶

Para este experimento se realiz un algoritmo gentico que intenta encontrar una ecuacin que llegue a un target especfico, utilizando solo un set de nmeros.

Para llevar a cabo esta implementacin, se desarrollo dos nuevas clases ChNode y ChTree, ambas hijos de Node y AbstractTree correpondientemente. En ChNode se guarda un set de datos que se pueden utilizar y se cambia la funcion de divisin por una floor_division que protege cuando se divide por cero.

A continuacin se muestra un ejemplo utilizando estas clases.
